Apakah di situs web ini membahas tentang air?

Situs web kami sudah memberikan sebuah bagan pada halaman homepage, yang membahas secara keseluruhan tentang air, dari sumber hingga cara mengolahnya.

Apakah di situs web ini memiliki form keluhan terkait air bersih?

Di situs web ini, terdapat commentar forum pada bagian footer di setiap halaman, disana akan ada pertanyaan "Apakah Anda ada kendala terkait akses air bersih di wilayah Anda?

Bagaimana caranya kembali ke halaman homepage?

Di setiap halaman, di bagian header terdapat icon gambar rumah yang mengarahkan ke halaman homepage dan di bagian footer terdapat navigation bar "Home", itu juga akan mengarahkan ke halaman homepage.

Apakah saat ini sudah bisa mengganti data profil?

Untuk saat ini, belum bisa mengatur data profil, dikarenakan ini masih dalam beta test User Experience, untuk update kedepannya akan kami usahakan membangun itu semua.

Apabila memiliki keluhan terkait situs web ini, dapat kemana?

Untuk ini, dapat langsung diajukan ke feeback page, untuk akses mudahnya terdapat di setiap page bagian footer pada navigation bar "Feedback", keluhan-keluhan ini akan kami terima sebagai bahan pengembangan situs web kami.
